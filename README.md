# BMI Calculator

## Overview

The BMI Calculator is a simple Python program that calculates a user's Body Mass Index (BMI) based on their weight and height. It also classifies the BMI into different health categories such as Underweight, Normal Weight, Overweight, or Obese.

## Features

* Calculates BMI using weight and height inputs.
* Displays the BMI value rounded to two decimal places.
* Categorizes BMI according to standard BMI ranges.
* Handles invalid inputs using exception handling.
* Prevents negative or zero values for weight and height.

## Technologies Used

* Python

## How It Works

1. Enter your weight in kilograms (kg).

2. Enter your height in meters (m).

3. The program calculates your BMI using the formula:

   **BMI = Weight (kg) / Height² (m²)**

4. The calculated BMI and corresponding category are displayed.

## BMI Categories

* Underweight: BMI less than 18.5
* Normal Weight: BMI between 18.5 and 24.9
* Overweight: BMI between 25 and 29.9
* Obese: BMI 30 or above
